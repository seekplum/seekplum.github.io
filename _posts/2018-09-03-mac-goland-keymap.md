## 常用快捷键
* 查看所有快捷键

> Preferences -> keymap

* 搜索文件内容

> shift + command + f

* 搜索文件名

> shift + command + o

* 复制选中内容

> command + d

* 删除当前行

> command + delete

* 格式化代码

> option + command + l

* 跳转到指定行

> command + l
