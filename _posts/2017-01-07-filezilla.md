## 安装filezilla

> sudo apt-get install filezilla

> ssh 连接服务器 默认端口为 22

## 装语言包

> sudo apt-get install filezilla-locales

> 安装完语言包就可以支持中文了

> 安装成功后，如果出现./filezilla/filezilla.xml、./filezilla/layout.xml等文件不允许些或者说不能够读，可能是权限问题。然后，把这些文件赋予合适的权限就成功了。
